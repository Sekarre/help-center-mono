# Help center monolith
Project simulating game support sites with features like: creating new issues, issues' management, chatting and basics like login/logout
## Technologies
- Java 17
- Spring Framework 5
  - Spring Boot
  - Spring Security  
  - Spring Data
  - Spring MVC  
  - Spring AOP
- WebSockets
- JUnit 5
- Maven
- Hibernate
- PostgreSQL, H2
- Lombok 
- MapStruct
